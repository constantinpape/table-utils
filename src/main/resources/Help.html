BigDataViewer Actions

Below keyboard short-cuts are available when BigDataViewer is the active window.

Ctrl + Left-click Toggle image segment selection
Crtl + Q >Unselect all image segments
Ctrl + S Iterate selection visualisation mode:
  • Dim not selected
  • Only show selected
  • Color selected bright yellow
  • Color selected bright yellow and dim not selected
Ctrl + L Shuffle colors of random LUT for categorical coloring
Ctrl + Shift + Left-click View image segment in 3D
